# Municipal Comparison Project

The main outputs of this noebook are 3 JSON files, found in the 'results' directory:
1. cities_to_features: all of the cities, and their corresponding predefined "interseting" features.
2. feature_categories_mapping: all of the "interesting" features, mapped to their corresponding categories. 
3. most_similar: all of the cities, mapped to their corresponding closest cities, as defined by various features.
